Current Badge Catalog

Last updated: 2026-03-23

This document lists the current claimable badge definitions in the app. It is based on the seeded badge catalog in web/badgeCatalog.js.

Important note:

  • The badge wall currently shows a larger 20-pin media shelf.
  • The live claimable badge catalog currently has 16 seeded badge definitions.
  • Today, those seeded definitions are attached to pin1 and pin2 assets. The other pins are available as media assets, but are not yet attached to seeded claim definitions.

Criteria Reference

  • Manual attestor approval: a trusted attestor records the badge claim directly for the agent.
  • Badge count check: the registry checks how many badges the agent already has.
  • Token balance check: the registry checks the connected wallet's balance on the configured token contract.
  • Oracle attendance attestation: the agent claims with a signed event-proof package from the configured oracle signer.
  • Agent reputation attestation: the agent claims with a signed attestation from a reputable agent, subject to the configured threshold.
