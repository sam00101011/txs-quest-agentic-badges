            Local live loop: run bun run deploy:local, then click
            Use Local Deployment. On localhost, the wallet picker also includes a
            built-in Local Dev Wallet for browser-side issuance against Anvil.
          

          

            Localhost admin writes: badge definition and asset registration are routed through the
            built-in Local Dev Wallet on chain 31337, even if you keep an
            injected wallet selected for claims or MPP payment.
          

          

            Paid live minting: run bun run mint:server, keep the contract wallet on
            localhost if you want, then connect with Tempo Connect or a wallet extension to approve the MPP charge.
          

          

            Reusable oracle proofs: run bun run oracle:server. The service evaluates
            wallet-age, protocol, portfolio, and internal-service badges only when the connected
            wallet actively tries to claim.
          

          

            On-demand payment proofs: run bun run x402:server. The service only evaluates
            a wallet's x402 and optional MPP payment history when that connected wallet actively tries
            to claim.
          

          

            Testnet deploys: copy config/testnet.example.json to
            config/testnet.json, fill in your RPC and proof URLs, then run
            bun run deploy:testnet.
